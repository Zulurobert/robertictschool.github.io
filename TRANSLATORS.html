TRANSLATORS

Translators are programs that are written and supplied by computer manufactures together with machines .Computer programs when written they are in human sensible form hence they is need to translate these programs into machine code i.e. from source program to object program

The mostly widely used translators used are the:
Compiler, interpreters and, assemblers

THE INTERPRETERS:

This is a translator program which translates and executes its source code in logical sequence as the program is being executed. For example the program has 50 lines, 1 line will be converted into machine form and then executed then line e.t.c. Interpreters are widely used in programming languages based on small computer because they are easier to use than compilers.

THE COMPILERS

This is a special translator program which translates high level languages into machine language for execution by the computer.

SOUCRCE CODE

These are program instruction written in high level programming languages that must be translated into machine language to be executed by computer.

THE ASSEMBLERS

This is a translator program which translate assembly language statement into machine Code .The machine is to generate for each source code instructions and involves mnemonic source code into machine code.

THE OBJECT CODE

The process of translating source into machine code is called object code

THE MACHINE LANGUAGUE

It is a programming language consisting of either 1 or 0 of binary codes

FILE ORGANISATION TERMS AND CONCEPT

computer system organizes data in hierarchy that start with:


• BIT
• BYTES
• FIELD
• RECORD
• FILE
• DATABASE

A Bit represents the smallest unit of data a computer can handle.

group of bits is called a BYTE represents a single character, which can be a, number or another symbol.

A Field is a group of characters in the word, a group of words, or a complete number, such as persons name or age.

A Record is a group of related fields, such as the student‘s name, the course taking, the date, enrolled.

A File is a group of records of the same type or related fields.

A Database is a group of related files.

An Entity is a person, place, things or events on which we maintain information. A order is a typical entity in a sales order file, which maintain information on the firm’s sales order.

An Attribute is a piece of information describing a particular entity.

A Key field is a field in a record that uniquely identifies instances of that record so that it can be retrieved, updated, or sorted.

Database management is simply the software that permits an organization to centralized data , manage them efficiently , and provide access to the stored data by applications programs .The DBMS acts as an interface between application programs and the and the physical data .

Logical view is the representation of data as they would appear to an application programmer or end user.

Physical view is the representation of data as they would actually be organized on the physical storage media.

Data Definition language the component of database management system that defines each data element as it appears in the database.

Data manipulation language a language associated with a database management system that end user and programmers use to manipulate data in the database.

Structured query language SQL

The standard data manipulation for rational database management system.

Data dictionary

An automated or manual tool for storing and organizing data / information about the data manipulation in database
