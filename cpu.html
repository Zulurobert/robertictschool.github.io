CPU

The brain of the computer. This is located on a circuit board inside the desktop or tower computer case

The CENTRAL PROCESSOR which is subdivided into:


i. The CONTROL UNIT (CU);
ii. The ARITHMETIC and LOGIC UNIT (ALU);
iii. The MAIN STORE (Primary Stora

The arrangement and control of the CPU varies from one computer to another. For example, some computers are controlled by a single processor whereas others may contain a number of processors, each controlling a different operation, e.g. Input/Output. This and other options give rise to the variety of computers that are available today. However, in all cases, the basic concept follows that of the Analytical Engine.

FUNCTIONAL COMPONENTS OF A COMPUTER SYSTEM

The Processor

The TWO components of the CPU that make up a processor are the Control Unit and the Arithmetic and Logic Unit (ALU).

Control unit

The control unit is a computer component which interprets instructions from the main storage, issues commands to all the elements of a computer: Input, ALU, main storage, Output and auxilliary storage.

Arithmetic & Logical Unit (ALU)

This is a component of the CPU whose purpose is to:
to carry out arithmetic operations on data, e.g. adding two numbers.
to carry out logical operations on data, e.g. comparing two numbers to find which one is larger than the other.

Main Memory

Memory refers to integrated circuits that store programs instructions and data that can be retrieved. The memory of a computer is the storage area for programs, along with its data while they are executing.
The purpose of main memory is:


• to store programs during their execution.
• to store data is being used by the current program
• to store the operating system.

Random Access Memory (RAM)

RAM is the name given to the integrated circuits, or chips that are used for main memory. It is called “Random access” because of the fact that any address in memory can be accessed at random, or in no particular order during a work session.

RAM is not a permanent storage place for information; it is active only when a computer is on.
When the computer is turned off, information is deleted from memory. Computer memory is measured in Kilobytes (KB), or Megabytes (MB) of information. (A byte is the amount of storage needed to hold one character). 1 KB = 1024 bytes and 1 MB = 1,048,576 bytes.

Speed of processors

The speed of processors is measured in Megaherz (MHz)

What is MHz?

It is a unit of measurement named after Heinrich R. Herz, a Germany Physicist, who first detected electromagnetic waves in 1883. 1 Herz (Hz) is equal to 1 cycle per second. Consequently Megaherz is a unit of measurement for indicating the frequency of one million electrical vibration cycles per second. A Gigaherz (GHz) is a unit of measurement for indicating the frequency of one billion electrical vibration cycles per second. Lets take a computer with the following processor speed: 300 MHz, this means that this computer can process 300 million instructions per second. These days, most processors range from 100MHz to over 2.0 GHz.

Read Only Memory (ROM)

This is the name given to chips that store information or instructions that do not change. For example, ROM is used to store the start up instructions and data used when a computer is first turned on.

Other types of ROM

• PROM – Programmable Read Only Memory
• EPROM – Erasable Programmable Read Only Memory
• EEPROM – Electrical Erasable Programmable Read Only Memory
• CD-ROM – Compact Disc Read Only Memory
• DVD-ROM - Digital Versatile Disc Read Only memory
